Send screenshot

You take a screen dump by pressing the “turn on” and the “lower sound” buttons at the same time. Successful taking a screen dump can be heard by a small click and seen by a small visualization of the screen dump taken. You can always send the last screen dump by mail by using the button “send screen” on the main screen. You can add a small explanation to the screen dump when you send it. First time you use the button “send screen” the mail address is empty and you need to fill it out. The next time it remembers your last e-mail. You can always change the e-mail address. You are not able to send older screen dumps. If you experience an error on the tablet, you can use screen dump to document the error and send it to support@tinymobilerobots.com
